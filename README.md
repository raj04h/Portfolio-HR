# Portfolio App 📱

A professional Android application built to showcase my journey, skills, and accomplishments as a developer. This app serves as a digital resume, providing an interactive and engaging way for recruiters and collaborators to explore my professional background.

---

## 🚀 Features

The application is structured into several key modules to provide a comprehensive view of my profile:

*   **Experience**: A detailed timeline of my professional work history.
*   **Projects**: A showcase of my key development projects with descriptions and tech stacks.
*   **Skills**: A categorized overview of my technical expertise and tools.
*   **Education**: Academic background and relevant certifications.
*   **Certificates**: A digital gallery of earned professional certifications.
*   **Interactive UI**: Smooth navigation and transitions between different sections using Jetpack Navigation.

---

## 🛠 Tech Stack & Architecture

This project follows modern Android development standards and best practices:

*   **Language**: [Kotlin](https://kotlinlang.org/) - 100% type-safe and concise.
*   **Architecture**: MVVM (Model-View-ViewModel) for clean separation of concerns and testability.
*   **UI Framework**: Material Design 3 for a modern and intuitive user interface.
*   **View Binding**: To safely interact with UI components.
*   **Jetpack Navigation**: For seamless fragment-to-fragment navigation.
*   **Lifecycle Components**: ViewModel and LiveData to handle data in a lifecycle-aware manner.

## ⚙️ Getting Started

### Prerequisites
*   Android Studio Ladybug (or later)
*   JDK 17
*   Android SDK 24+

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/raj04h/Portfolio-HR.git
    ```
2.  Open the project in **Android Studio**.
3.  Let the Gradle sync complete.
4.  Run the app on an emulator or physical device.
